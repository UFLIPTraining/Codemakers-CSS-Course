CSS Text

Discover how to style text in CSS, including font properties, text alignment, spacing, and decoration.

Text Properties

CSS offers a variety of properties to style text, such as color, font-size, font-weight, text-align, and text-decoration. Here is an example:

/* styles.css */
.text-example {
  color: #00cdff;
  font-size: 20px;
  font-weight: bold;
  text-align: center;
  text-decoration: underline;
}

Example Exercise

Try creating a paragraph with different text properties:

<!DOCTYPE html>

  <title>CSS Text Exercise</title>
  <link rel="stylesheet" href="styles.css">

</body>

Example 1: Text Shadow

Add a shadow effect to your text using the text-shadow property:

/* styles.css */
.text-shadow-example {
  color: #1c75bc;
  font-size: 24px;
  text-shadow: 2px 2px 5px rgba(0, 0, 0, 0.3);
}

Try creating a paragraph with a text shadow:

<p class="text-shadow-example">This text has a shadow effect.</p>

Example 2: Letter Spacing

Adjust the spacing between characters using the letter-spacing property:

/* styles.css */
.letter-spacing-example {
  color: #febd1d;
  font-size: 22px;
  letter-spacing: 2px;
}

Try creating a paragraph with letter spacing:

<p class="letter-spacing-example">This text has increased letter spacing.</p>

Example 3: Line Height

Control the height of lines of text using the line-height property:

/* styles.css */
.line-height-example {
  color: #00cdff;
  font-size: 20px;
  line-height: 1.8;
}

Try creating a paragraph with increased line height:

<p class="line-height-example">This text has increased line height for better readability.</p>

CSS Fonts

Learn how to use and style different fonts in CSS, including web-safe fonts, custom fonts, and Google Fonts.

Font Properties

CSS allows you to use a variety of fonts by specifying font families, sizes, and styles. Here is an example:

/* styles.css */
.font-example {
  font-family: 'Arial', sans-serif;
  font-size: 18px;
  font-style: italic;
  font-weight: 600;
}

Using Google Fonts

You can use custom fonts from Google Fonts by importing them into your CSS. Here is an example:

/* Add this at the top of your styles.css */
@import url('https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap');

/* styles.css */
.google-font-example {
  font-family: 'Roboto', sans-serif;
  font-size: 18px;
  font-weight: 400;
}

Example 1: Custom Fonts with @font-face

Use custom fonts by defining them with the @font-face rule:

/* styles.css */
@font-face {
  font-family: 'CustomFont';
  src: url('fonts/CustomFont.woff2') format('woff2'),
       url('fonts/CustomFont.woff') format('woff');
  font-weight: normal;
  font-style: normal;
}

.custom-font-example {
  font-family: 'CustomFont', sans-serif;
  font-size: 18px;
}

Try creating a paragraph using a custom font:

<p class="custom-font-example">This text uses a custom font.</p>

Example 2: Font Variants

Use different font variants like small-caps and ligatures:

/* styles.css */
.font-variant-example {
  font-family: 'Times New Roman', serif;
  font-size: 20px;
  font-variant: small-caps;
}

Try creating a paragraph with a font variant:

<p class="font-variant-example">This text uses small-caps font variant.</p>

Example 3: Font Stacks

Use font stacks to specify a list of fallback fonts:

/* styles.css */
.font-stack-example {
  font-family: 'Helvetica Neue', Arial, sans-serif;
  font-size: 18px;
}

Try creating a paragraph with a font stack:

<p class="font-stack-example">This text uses a font stack.</p>
