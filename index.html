About This Course

This course is designed to introduce you to the world of CSS, providing a comprehensive guide from the basics to advanced topics. Each section focuses on different aspects of CSS to help you build beautiful and responsive web pages. This course will serve as your HTML/CSS playground, aiming to equip you with the knowledge and skills to enhance your web development capabilities.

How To

Installing Visual Studio Code

  1. Go to the Visual Studio Code website: https://code.visualstudio.com/
  2. Click on the download button for your operating system (Windows, macOS, or Linux).
  3. Once the download is complete, open the installer and follow the installation instructions.

Creating and Referencing a CSS Style Sheet

Follow these steps to create a CSS style sheet and reference it in your HTML file:

  1. Open Visual Studio Code.
  2. Create a new folder for your project.
  3. Inside the project folder, create a new file and name it styles.css.
  4. Add some CSS rules to the styles.css file. For example:
/* styles.css */
body {
  font-family: Arial, sans-serif;
  margin: 0;
  padding: 0;
}

header {
  background-color: #267738;
  color: white;
  text-align: center;
  padding: 10px 0;
}

nav {
  background-color: #333;
}

nav ul {
  list-style-type: none;
  padding: 0;
}

nav ul li {
  display: inline;
  margin-right: 10px;
}

nav ul li a {
  color: white;
  text-decoration: none;
  padding: 5px 10px;
}

nav ul li a:hover {
  background-color: #267738;
}

main {
  padding: 20px;
}
  1. Create a new file in the project folder and name it index.html.
  2. Add the following HTML code to the index.html file to reference the CSS style sheet:
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>My Project</title>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <header>
    <h1>Welcome to My Project</h1>
  </header>
  <nav>
    <ul>
      <li><a href="#">Home</a></li>
      <li><a href="#">About</a></li>
      <li><a href="#">Contact</a></li>
    </ul>
  </nav>
  <main>
    <p>This is the main content area.</p>
  </main>
</body>
</html>

Save both files and open the index.html file in a web browser to see your styled page.

Using the Capstone Project from the HTML Course

Throughout this course, you will be building upon the capstone project you created in the HTML course. Make sure to use the HTML structure you developed previously and apply the new CSS skills you learn in this course.

Capstone Project

For the capstone project, you will rebuild the blog you created in the HTML Course. Ensure you don't add any inline styling and stick to CSS for all your styling needs. If you need to rebuild the HTML capstone, that's okay. This project will help you consolidate all the CSS skills you will learn in this course.

Steps for Styling Your Blog

  1. Review Your HTML Structure: Ensure your blog uses semantic HTML elements, ARIA roles for accessibility, and is SEO optimized.
  2. Create and Link a CSS File: Create a new CSS file (e.g., styles.css) and link it to your HTML file.
  3. Apply Base Styles: Set up base styles, including resetting margins and paddings, setting a font family, and defining general styles for the body and main container.
  4. Style the Header: Set background color, padding, and typography for the header.
  5. Style the Main Content: Focus on typography, spacing, and layout for articles and sections. Ensure images are responsive.
  6. Style Navigation and Footer: Ensure consistent styling for navigation links and footer content.
  7. Add Responsive Design: Use media queries to adjust styles for different screen sizes.

Exercises

  1. Review and refactor your HTML blog post to ensure it uses semantic elements, ARIA attributes, and is SEO optimized. Make any necessary changes to improve the structure and accessibility.
  2. Create and link a CSS file to your HTML blog post. Apply base styles to set up a consistent font, background color, and basic layout for the body and main container.
  3. Style your blog's header, main content (including articles, sections, and images), navigation bar, and footer. Add responsive design using media queries to ensure your blog looks good on different screen sizes.
