About This Course

This course is designed to introduce you to the world of CSS, providing a comprehensive guide from the basics to advanced topics. Each section focuses on different aspects of CSS to help you build beautiful and responsive web pages. This course will serve as your HTML/CSS playground, aiming to equip you with the knowledge and skills to enhance your web development capabilities.

Setting Up GitHub

Creating a GitHub Account

  1. Go to the GitHub website: https://github.com/
  2. Click on the "Sign up" button and fill in the required information to create your account.

Forking the Repository

  1. Log in to your GitHub account.
  2. Go to the CSS Course repository: https://github.com/UFLIPTraining/Codemakers-CSS-Course
  3. Click on the "Fork" button at the top-right corner of the page to create a copy of the repository in your own GitHub account.

Disabling the Template Repository

  1. Go to the settings of your forked repository on GitHub.
  2. Scroll down to the "Template repository" section.
  3. Uncheck the "Template repository" checkbox.
  4. Click "Save changes".

Collaborating on the Repository

  1. Go to the settings of your forked repository on GitHub.
  2. Click on the "Manage access" tab in the left sidebar.
  3. Click on the "Invite a collaborator" button.
  4. Enter the username JMartinUF and click "Add collaborator".
  5. Click "Send invitation" to invite JMartinUF to collaborate on your repository.

Steps for Styling Your Blog

  1. Review Your HTML Structure: Ensure your blog uses semantic HTML elements, ARIA roles for accessibility, and is SEO optimized.
  2. Create and Link a CSS File: Create a new CSS file (e.g., styles.css) and link it to your HTML file.
  3. Apply Base Styles: Set up base styles, including resetting margins and paddings, setting a font family, and defining general styles for the body and main container.
  4. Style the Header: Set background color, padding, and typography for the header.
  5. Style the Main Content: Focus on typography, spacing, and layout for articles and sections. Ensure images are responsive.
  6. Style Navigation and Footer: Ensure consistent styling for navigation links and footer content.
  7. Add Responsive Design: Use media queries to adjust styles for different screen sizes.

Exercises

  1. Review and refactor your HTML blog post to ensure it uses semantic elements, ARIA attributes, and is SEO optimized. Make any necessary changes to improve the structure and accessibility.
  2. Create and link a CSS file to your HTML blog post. Apply base styles to set up a consistent font, background color, and basic layout for the body and main container.
  3. Style your blog's header, main content (including articles, sections, and images), navigation bar, and footer. Add responsive design using media queries to ensure your blog looks good on different screen sizes.
