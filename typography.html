Typography

Typography involves styling text to make it readable and visually appealing. Key properties include font-family, font-size, font-weight, line-height, and text-align.

Font Properties

/* CSS */
.typography-example {
  font-family: 'Arial', sans-serif;
  font-size: 18px;
  font-weight: bold;
  line-height: 1.5;
  text-align: center;
}
This is an example of styled text.

Inspect the element above and try changing its font-family, font-size, font-weight, line-height, and text-align properties.

Exercises

  1. Create a new element with a class typography-example-2 and apply a different font family, font size, font weight, and line height to it.
  2. Change the text alignment of the typography-example element to justify and observe the changes.
  3. Add a new CSS class called text-decoration-example that applies an underline to the text. Apply this class to one of the existing elements.

Using Google Fonts

You can use custom fonts from Google Fonts by importing them into your CSS. Here is an example:

/* Add this at the top of your styles.css */
@import url('https://fonts.googleapis.com/css2?family=Roboto:wght@400;700&display=swap');

/* CSS */
.google-font-example {
  font-family: 'Roboto', sans-serif;
  font-size: 18px;
  font-weight: 400;
}
This text uses the Roboto font from Google Fonts.

Inspect the element above and try changing its font-family, font-size, and font-weight properties.

Exercises

  1. Create a new element with a class google-font-example-2 and apply a different Google font with a different font weight to it.
  2. Change the font size of the google-font-example element to 24px and observe the changes.
  3. Add a new CSS class called font-weight-example that sets the font weight to bold. Apply this class to one of the existing elements.

Text Decoration

The text-decoration property adds decorations to text. Common values include none, underline, overline, and line-through.

/* CSS */
.text-decoration-example {
  font-size: 18px;
}

.underline {
  text-decoration: underline;
}

.overline {
  text-decoration: overline;
}

.line-through {
  text-decoration: line-through;
}
This text is underlined.
This text has an overline.
This text has a line-through.

Inspect the elements above and try changing their text-decoration properties.

Exercises

  1. Create a new element with a class text-decoration-example-2 and apply an overline to the text.
  2. Remove the text decoration from the underline element and observe the changes.
  3. Add a new CSS class called text-decoration-example-3 that applies a line-through to the text. Apply this class to one of the existing elements.

Text Transform

The text-transform property controls the capitalization of text. Common values include none, capitalize, uppercase, and lowercase.

/* CSS */
.text-transform-example {
  font-size: 18px;
}

.capitalize {
  text-transform: capitalize;
}

.uppercase {
  text-transform: uppercase;
}

.lowercase {
  text-transform: lowercase;
}
this text is capitalized.
this text is uppercase.
THIS TEXT IS LOWERCASE.

Inspect the elements above and try changing their text-transform properties.

Exercises

  1. Create a new element with a class text-transform-example-2 and apply uppercase transformation to it.
  2. Change the text of the capitalize element to "this text is capitalized" and observe the changes.
  3. Add a new CSS class called text-transform-example-3 that applies lowercase transformation to the text. Apply this class to one of the existing elements.
