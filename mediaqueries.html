Media Queries

Media queries allow you to apply CSS rules based on the size of the viewport. This helps in making your website responsive to different screen sizes.

/* CSS */
.media-query-example {
  background-color: #1c75bc;
  color: white;
  padding: 20px;
  text-align: center;
}

@media (max-width: 768px) {
  .media-query-example {
    background-color: #00cdff;
  }
}

@media (max-width: 480px) {
  .media-query-example {
    background-color: #febd1d;
  }
}
Resize the browser to see the background color change.

Inspect the element above and try changing the media queries.

Exercises

  1. Create a new media query for (min-width: 1024px) and change the background color of the .media-query-example to lightgreen.
  2. Add a new CSS rule within the existing media queries to change the font size of the .media-query-example text based on the viewport width.
  3. Modify the existing media queries to include a padding change for the .media-query-example element. Make the padding 40px for viewports less than 768px and 60px for viewports less than 480px.

Units via Print CSS

Print CSS allows you to style your content for print media. Units like cm, mm, in, and pt are used for print styling.

/* CSS */
@media print {
  body {
    font-size: 12pt;
    line-height: 1.5;
  }
  .print-example {
    margin: 2cm;
    padding: 1cm;
    border: 1mm solid black;
  }
}
This is styled for print media.

Inspect the element above and try changing the units in the print media query.

Exercises

  1. Create a new print media query that changes the text color of the .print-example element to gray and the background color to lightyellow.
  2. Adjust the existing print media query to change the border width of the .print-example element to 2mm and the padding to 1.5cm.
  3. Add a new CSS rule within the print media query to change the font size of the body to 14pt and observe the changes.
