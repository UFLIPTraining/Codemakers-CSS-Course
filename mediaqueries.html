Media Queries

Media queries allow you to apply CSS rules based on the size of the viewport. This helps in making your website responsive to different screen sizes.

/* CSS */
.media-query-example {
  background-color: #1c75bc;
  color: white;
  padding: 20px;
  text-align: center;
}

@media (max-width: 768px) {
  .media-query-example {
    background-color: #00cdff;
  }
}

@media (max-width: 480px) {
  .media-query-example {
    background-color: #febd1d;
  }
}
Resize the browser to see the background color change.

Units via Print CSS

Print CSS allows you to style your content for print media. Units like cm, mm, in, and pt are used for print styling.

/* CSS */
@media print {
  body {
    font-size: 12pt;
    line-height: 1.5;
  }
  .print-example {
    margin: 2cm;
    padding: 1cm;
    border: 1mm solid black;
  }
}
This is styled for print media.

Inspect the element above and try changing the units in the print media query.
