Responsive Web Design (RWD) Intro

Learn the basics of responsive web design, including the principles and techniques used to create responsive layouts.

Responsive Layout

To create a simple responsive layout, you can use the following example:

/* styles.css */
.responsive-container {
    display: flex;
    flex-wrap: wrap;
    gap: 20px;
}

.responsive-item {
    flex: 1 1 100px;
    background-color: #1c75bc;
    color: white;
    padding: 20px;
    box-sizing: border-box;
}

Start by creating a simple responsive layout:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>RWD Example</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <div class="responsive-container">
        <div class="responsive-item">Item 1</div>
        <div class="responsive-item">Item 2</div>
        <div class="responsive-item">Item 3</div>
    </div>
</body>
</html>

Example 1: Responsive Flexbox Layout

Using Flexbox to create a responsive layout:

/* styles.css */
.flexbox-container {
    display: flex;
    flex-wrap: wrap;
    gap: 10px;
}

.flexbox-item {
    flex: 1 1 calc(33.33% - 20px);
    background-color: #febd1d;
    padding: 20px;
    text-align: center;
    box-sizing: border-box;
}

@media (max-width: 600px) {
    .flexbox-item {
        flex: 1 1 100%;
    }
}

Create a responsive Flexbox layout:

<div class="flexbox-container">
    <div class="flexbox-item">Flex Item 1</div>
    <div class="flexbox-item">Flex Item 2</div>
    <div class="flexbox-item">Flex Item 3</div>
</div>

RWD Viewport

Discover how to use the viewport meta tag and media queries to create responsive designs.

Viewport Meta Tag

The viewport meta tag is essential for responsive design. Here is an example:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Media Queries

Media queries allow you to apply CSS rules based on the size of the viewport. For example:

/* styles.css */
.viewport-example {
    background-color: #1c75bc;
    color: white;
    padding: 20px;
    text-align: center;
}

@media (max-width: 600px) {
    .viewport-example {
        background-color: #00cdff;
    }
}

Use the viewport meta tag and media queries to make a layout responsive:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Viewport and Media Queries</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <div class="viewport-example">
        Responsive Design Example
    </div>
</body>
</html>

RWD Grid View

Explore how to create responsive grid layouts using CSS Grid and Flexbox.

CSS Grid

CSS Grid is a powerful tool for creating responsive layouts. Here is an example:

/* styles.css */
.grid-container {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(100px, 1fr));
    gap: 20px;
}

.grid-item {
    background-color: #1c75bc;
    color: white;
    padding: 20px;
    text-align: center;
}

Create a responsive grid layout:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Responsive Grid</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <div class="grid-container">
        <div class="grid-item">Grid Item 1</div>
        <div class="grid-item">Grid Item 2</div>
        <div class="grid-item">Grid Item 3</div>
    </div>
</body>
</html>

Example 1: Grid Template Areas

Create a responsive layout using grid template areas:

/* styles.css */
.grid-template-areas-container {
    display: grid;
    grid-template-areas:
        'header header'
        'sidebar main'
        'footer footer';
    gap: 10px;
}

.grid-header {
    grid-area: header;
    background-color: #1c75bc;
    color: white;
    padding: 20px;
    text-align: center;
}

.grid-sidebar {
    grid-area: sidebar;
    background-color: #00cdff;
    color: black;
    padding: 20px;
    text-align: center;
}

.grid-main {
    grid-area: main;
    background-color: #febd1d;
    color: black;
    padding: 20px;
    text-align: center;
}

.grid-footer {
    grid-area: footer;
    background-color: #333;
    color: white;
    padding: 20px;
    text-align: center;
}

@media (max-width: 600px) {
    .grid-template-areas-container {
        grid-template-areas:
            'header'
            'main'
            'sidebar'
            'footer';
    }
}

Create a responsive layout using grid template areas:

<div class="grid-template-areas-container">
    <div class="grid-header">Header</div>
    <div class="grid-sidebar">Sidebar</div>
    <div class="grid-main">Main</div>
    <div class="grid-footer">Footer</div>
</div>

RWD Media Queries

Learn how to use media queries to create responsive designs that adapt to different screen sizes.

Responsive Media Queries

Implement media queries to adapt the layout:

/* styles.css */
.media-query-example {
    background-color: #1c75bc;
    color: white;
    padding: 20px;
    text-align: center;
}

@media (max-width: 768px) {
    .media-query-example {
        background-color: #00cdff;
    }
}

@media (max-width: 480px) {
    .media-query-example {
        background-color: #febd1d;
    }
}

Implement media queries to adapt the layout:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Responsive Media Queries</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <div class="media-query-example">
        Media Query Example
    </div>
</body>
</html>

Example 1: Change Font Size with Media Queries

Adjust the font size based on screen width:

/* styles.css */
.font-size-example {
    font-size: 18px;
}

@media (max-width: 768px) {
    .font-size-example {
        font-size: 16px;
    }
}

@media (max-width: 480px) {
    .font-size-example {
        font-size: 14px;
    }
}

Adjust the font size based on screen width:

<p class="font-size-example">This text changes size based on screen width.</p>
