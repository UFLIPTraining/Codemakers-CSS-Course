CSS Navigation Bars

Discover how to create and style navigation bars in CSS, including horizontal and vertical menus.

Horizontal Navigation Bar

To create a horizontal navigation bar, you can use the following example: 

/* styles.css */
.nav-bar ul {
    display: flex;
    list-style: none;
    padding: 0;
    margin: 0;
}
.nav-bar ul li {
    margin: 0 10px;
}
.nav-bar ul li a {
    color: #fff;
    text-decoration: none;
    font-weight: bold;
}

Try creating a horizontal navigation bar:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>CSS Navigation Bar Exercise</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <nav class="nav-bar">
        <ul>
            <li><a href="#">Home</a></li>
            <li><a href="#">About</a></li>
            <li><a href="#">Services</a></li>
            <li><a href="#">Contact</a></li>
        </ul>
    </nav>
</body>
</html>

Vertical Navigation Bar

To create a vertical navigation bar, you can use the following example: 

/* styles.css */
.vertical-nav-bar ul {
    list-style: none;
    padding: 0;
    margin: 0;
}
.vertical-nav-bar ul li {
    margin: 10px 0;
}
.vertical-nav-bar ul li a {
    color: #fff;
    text-decoration: none;
    padding: 10px;
    display: block;
    transition: background 0.3s, color 0.3s;
}
.vertical-nav-bar ul li a:hover {
    background: #1c75bc;
    color: #fff;
}

Try creating a vertical navigation bar:

        </ul>

CSS Buttons

Learn how to style buttons in CSS, including different states, sizes, and types.

Button States

To style a button with different states (e.g., hover, active), you can use the following example: 

/* styles.css */
.button {
    background-color: #1c75bc;
    color: #fff;
    border: none;
    padding: 10px 20px;
    cursor: pointer;
    transition: background-color 0.3s;
}
.button:hover {
    background-color: #00cdff;
}

Try styling a button with different states:

<!DOCTYPE html>

    <title>CSS Buttons Exercise</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>

</body>
</html>

CSS Forms

Discover how to style forms in CSS, including input fields, labels, and validation.

Styled Form

To style a form, you can use the following example:

/* styles.css */
.form-example {
    display: flex;
    flex-direction: column;
    max-width: 300px;
    margin: 0 auto;
}
.form-example label {
    margin-bottom: 5px;
    color: #bebcbc;
}
.form-example input {
    margin-bottom: 10px;
    padding: 10px;
    border: 1px solid #444;
    background-color: #232323;
    color: #fff;
}
.form-example button {
    background-color: #1c75bc;
    color: #fff;
    border: none;
    padding: 10px 20px;
    cursor: pointer;
    transition: background-color 0.3s;
}
.form-example button:hover {
    background-color: #00cdff;
}

Try styling a form:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>CSS Forms Exercise</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <form class="form-example">
        <label for="name">Name:</label>
        <input type="text" id="name" name="name">
        <label for="email">Email:</label>
        <input type="email" id="email" name="email">
        <button type="submit">Submit</button>
    </form>
</body>
</html>

CSS Modals

Learn how to create and style modals in CSS for displaying overlay content.

Styled Modal

To create and style a modal, you can use the following example:

/* styles.css */
.modal {
    display: none;
    position: fixed;
    top: 0;
    left: 0;
    width: 100%;
    height: 100%;
    background-color: rgba(0, 0, 0, 0.7);
    justify-content: center;
    align-items: center;
}
.modal-content {
    background-color: #232323;
    padding: 20px;
    border-radius: 5px;
    text-align: center;
    color: #fff;
}
.close-button {
    color: #fff;
    float: right;
    font-size: 1.5em;
    cursor: pointer;
}

Try creating a modal:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>CSS Modals Exercise</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <button id="open-modal">Open Modal</button>
    <div id="modal" class="modal">
        <div class="modal-content">
            <span class="close-button">×</span>
            <p>This is a modal!</p>
        </div>
    </div>
    <script>
        document.getElementById('open-modal').addEventListener('click', function() {
            document.getElementById('modal').style.display = 'block';
        });
        document.querySelector('.close-button').addEventListener('click', function() {
            document.getElementById('modal').style.display = 'none';
        });
    </script>
</body>
</html>
