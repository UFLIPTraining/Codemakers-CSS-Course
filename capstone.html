Capstone Project: Styling Your Blog

In this capstone project, you will apply the CSS skills you've learned to style a blog post created with HTML. The HTML blog post should use semantic elements, ARIA attributes, and be SEO optimized.

Step 1: Review Your HTML Structure

Ensure your HTML blog post is well-structured, using semantic tags like <header>, <article>, <section>, <nav>, and <footer>. Also, include ARIA roles and attributes where necessary to improve accessibility.

Step 2: Create a CSS File

Create a new CSS file (e.g., styles.css) in your project directory. Link this CSS file to your HTML blog post by adding the following line within the <head> section of your HTML file:

<link rel="stylesheet" href="styles.css">

Step 3: Apply Base Styles

Start by adding some base styles to your CSS file. These can include resetting margins and paddings, setting a base font, and defining general styles for the body and main container.

/* styles.css */
body {
  margin: 0;
  padding: 0;
  font-family: Arial, sans-serif;
  line-height: 1.6;
  background-color: #f4f4f4;
}

.container {
  max-width: 1200px;
  margin: 0 auto;
  padding: 20px;
}

Step 4: Style Your Header

Style the header section of your blog. This can include setting a background color, padding, and typography for the header elements.

/* styles.css */
header {
  background: #333;
  color: #fff;
  padding: 20px 0;
  text-align: center;
}

header h1 {
  margin: 0;
  font-size: 2em;
}

Step 5: Style the Main Content

Apply styles to the main content of your blog, including articles, sections, and images. Focus on typography, spacing, and layout.

/* styles.css */
article {
  background: #fff;
  margin: 20px 0;
  padding: 20px;
  border-radius: 5px;
  box-shadow: 0 0 10px rgba(0, 0, 0, 0.1);
}

article h2 {
  color: #333;
}

article p {
  line-height: 1.6;
}

img {
  max-width: 100%;
  height: auto;
}

Step 6: Style Navigation and Footer

Style your navigation bar and footer to ensure a consistent look and feel throughout your blog.

/* styles.css */
nav ul {
  list-style: none;
  padding: 0;
}

nav ul li {
  display: inline;
  margin-right: 10px;
}

nav ul li a {
  text-decoration: none;
  color: #333;
}

footer {
  background: #333;
  color: #fff;
  text-align: center;
  padding: 10px 0;
}

Step 7: Add Responsive Design

Use media queries to make your blog responsive, ensuring it looks good on different screen sizes.

/* styles.css */
@media (max-width: 768px) {
  .container {
    padding: 10px;
  }

  nav ul li {
    display: block;
    margin: 5px 0;
  }

  header h1 {
    font-size: 1.5em;
  }
}

Exercises

  1. Review and refactor your HTML blog post to ensure it uses semantic elements, ARIA attributes, and is SEO optimized. Make any necessary changes to improve the structure and accessibility.
  2. Create and link a CSS file to your HTML blog post. Apply base styles to set up a consistent font, background color, and basic layout for the body and main container.
  3. Style your blog's header, main content (including articles, sections, and images), navigation bar, and footer. Add responsive design using media queries to ensure your blog looks good on different screen sizes.
