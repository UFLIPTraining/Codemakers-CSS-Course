CSS Images

Learn how to style images in CSS, including responsive images, image borders, and image hover effects.

Image Styling

To style an image with borders and hover effects, you can use the following example: 

/* styles.css */
.image-example {
    max-width: 100%;
    height: auto;
    border: 5px solid #1c75bc;
    transition: transform 0.3s;
}
.image-example:hover {
    transform: scale(1.1);
}

Try styling an image with borders and hover effects:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>CSS Images Exercise</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <img src="example.jpg" alt="Example Image" class="image-example">
</body>
</html>

Exercises

  1. Change the border color of the image to a different color and add a box shadow effect.
  2. Add a grayscale filter to the image and change it back to color on hover.
  3. Make the image rotate slightly on hover using CSS transforms.

CSS Videos

Discover how to style and control videos in CSS, including responsive videos and custom controls.

Video Styling

To style a video element, you can use the following example:

/* styles.css */
.video-example {
    max-width: 100%;
    height: auto;
    border: 3px solid #00cdff;
    border-radius: 10px;
}

Try styling a video element:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>CSS Videos Exercise</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <video class="video-example" controls>
        <source src="example.mp4" type="video/mp4">
        Your browser does not support the video tag.
    </video>
</body>
</html>

Exercises

  1. Add a box shadow to the video element.
  2. Change the border-radius of the video to a different value and add a transition effect for smooth corners.
  3. Make the video element responsive by setting its width to a percentage of its container's width.
