Introduction to CSS

Learn the basics of CSS, including its history and core concepts.

History of CSS

CSS was first proposed by HÃ¥kon Wium Lie on October 10, 1994, and it was officially adopted as a W3C recommendation in December 1996. Since then, CSS has undergone several revisions:

  • CSS1 (1996): Introduced basic features like font properties, text alignment, and simple box model properties.
  • CSS2 (1998): Added more sophisticated layouts, including positioning and z-index.
  • CSS3 (2001): Introduced modules for easier updates and added features like media queries, selectors, and enhanced styling capabilities.

Why CSS Was Created

Before CSS, web developers used HTML tags like <font> and <center> to style web pages. This made HTML documents bulky and difficult to maintain. CSS was created to:

  • Separate content from presentation, making HTML cleaner and more semantic.
  • Provide more styling options, allowing for sophisticated and flexible styling.
  • Enable responsive design, ensuring web pages look good on different devices and screen sizes.

How CSS Works

CSS works by associating style rules with HTML elements. These rules are defined in CSS files or within the <style> tag in an HTML document. A CSS rule consists of a selector and a declaration block: 

selector {
  property: value;
}

Example:

body {
  background-color: lightblue;
}
h1 {
  color: navy;
  margin-left: 20px;
}

CSS Selectors

Selectors are used to target HTML elements for styling. Here are some common selectors:

  • Type Selector: Selects all elements of a given type. 
    p {
  color: blue;
}
  • Class Selector: Selects elements with a specific class attribute. 
    .intro {
  font-size: 20px;
}
  • ID Selector: Selects a single element with a specific id attribute. 
    #header {
  background-color: yellow;
}
  • Attribute Selector: Selects elements with a specific attribute. 
    a[target="_blank"] {
  color: red;
}
  • Descendant Selector: Selects elements that are descendants of another element. 
    div p {
  margin: 10px;
}

Example HTML Document with Basic CSS

Here is a semantic HTML document that you can use to practice writing CSS: 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First CSS Page</title>
    <link rel="stylesheet" href="css/styles.css">
</head>
<body>
    <header>
        <h1>Welcome to My Website</h1>
        <nav>
            <ul>
                <li><a href="#">Home</a></li>
                <li><a href="#">About</a></li>
                <li><a href="#">Services</a></li>
                <li><a href="#">Contact</a></li>
            </ul>
        </nav>
    </header>
    <main>
        <section>
            <h2>About Me</h2>
            <p>Hello! I am learning CSS to build beautiful websites.</p>
        </section>
        <section>
            <h2>My Services</h2>
            <p>I offer web development and design services.</p>
        </section>
    </main>
    <footer>
        <p>&copy; 2024 My Website. All rights reserved.</p>
    </footer>
</body>
</html>
