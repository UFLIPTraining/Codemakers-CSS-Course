CSS Positioning

Discover the different positioning techniques in CSS, including static, relative, absolute, fixed, and sticky positioning.

Example Exercise

Try using different positioning techniques:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>CSS Positioning Exercise</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <div class="static-position">Static Position</div>
    <div class="relative-position">Relative Position</div>
    <div class="absolute-position">Absolute Position</div>
    <div class="fixed-position">Fixed Position</div>
    <div class="sticky-position">Sticky Position</div>
</body>
</html>
