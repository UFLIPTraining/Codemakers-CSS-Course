Display

The display property determines how an element is displayed. Common values are block, inline, and inline-block.

/* CSS */
.block {
  display: block;
  width: 100%;
  background-color: lightblue;
  padding: 10px;
}

.inline {
  display: inline;
  background-color: lightgreen;
  padding: 10px;
}

.inline-block {
  display: inline-block;
  background-color: lightcoral;
  padding: 10px;
}

Display: None

Use display: none; to hide an element completely.

/* CSS */
.hidden {
  display: none;
}
This element is hidden.

Try adding display: block; to the .hidden class in the inspector to make it visible.

Display: Flex

Use display: flex; to create a flex container.

/* CSS */
.flex-container {
  display: flex;
  justify-content: space-around;
  background-color: lightgray;
}

.flex-item {
  background-color: #1c75bc;
  color: white;
  padding: 10px;
  margin: 5px;
}

Position

The position property specifies the type of positioning method used for an element. Common values are static, relative, absolute, fixed, and sticky.

/* CSS */
.static-position {
  position: static;
}

.relative-position {
  position: relative;
  top: 20px;
  left: 20px;
}

.absolute-position {
  position: absolute;
  top: 50px;
  left: 50px;
}

.fixed-position {
  position: fixed;
  top: 10px;
  right: 10px;
}

.sticky-position {
  position: -webkit-sticky;
  position: sticky;
  top: 0;
  background-color: yellow;
  padding: 50px;
  font-size: 20px;
}

Fixed Footer

A footer that stays fixed at the bottom of the page.

/* CSS */
.fixed-footer {
  position: fixed;
  bottom: 0;
  width: 100%;
  background-color: #333;
  color: white;
  text-align: center;
  padding: 10px;
}
This is a fixed footer.

Absolute Positioned Element

An element positioned absolutely inside a relatively positioned container.

/* CSS */
.relative-container {
  position: relative;
  background-color: lightgray;
  padding: 50px;
}

.absolute-element {
  position: absolute;
  top: 10px;
  right: 10px;
  background-color: #1c75bc;
  color: white;
  padding: 10px;
