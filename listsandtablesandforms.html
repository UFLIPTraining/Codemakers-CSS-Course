CSS Lists

Style lists in CSS, including ordered lists, unordered lists, and custom list styles.

Ordered and Unordered Lists

CSS allows you to style both ordered and unordered lists with various properties such as list-style-type, list-style-position, and list-style-image. Here is an example: 


/* styles.css */
.list-example ul {
    list-style-type: square;
    padding-left: 20px;
}
.list-example ol {
    list-style-type: decimal;
    padding-left: 20px;
}

Example Exercise

Try creating an ordered and unordered list with custom styles:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>CSS Lists Exercise</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <div class="list-example">
        <ul>
            <li>List Item 1</li>
            <li>List Item 2</li>
            <li>List Item 3</li>
        </ul>
        <ol>
            <li>List Item 1</li>
            <li>List Item 2</li>
            <li>List Item 3</li>
        </ol>
    </div>
</body>
</html>

Exercises

  1. Change the bullet points of the unordered list to circles.
  2. Use a custom image as the list item marker for the unordered list.
  3. Style the ordered list to use Roman numerals instead of decimal numbers.

CSS Tables

Learn how to style tables in CSS, including borders, spacing, and responsive tables.

Basic Table Styling

CSS allows you to style tables using properties like border, padding, and text-align. Here is an example: 


/* styles.css */
.table-example table {
    width: 100%;
    border-collapse: collapse;
}
.table-example th, .table-example td {
    border: 1px solid #00cdff;
    padding: 8px;
    text-align: left;
}
.table-example th {
    background-color: #232323;
    color: #fff;
}

  3. Add padding and change the text alignment for table headers and cells.

CSS Forms

Discover how to style forms in CSS, including input fields, labels, and validation.

Basic Form Styling

CSS allows you to style forms using properties like padding, margin, and border. Here is an example: 


/* styles.css */
.form-example input[type="text"], .form-example input[type="email"], .form-example input[type="password"] {
    width: 100%;
    padding: 10px;
    margin: 5px 0;
    border: 1px solid #00cdff;
    border-radius: 4px;
}
.form-example label {
    display: block;
    margin-bottom: 5px;
    font-weight: bold;
}
.form-example button {
    padding: 10px 15px;
    background-color: #00cdff;
    color: #fff;
    border: none;
    border-radius: 4px;
    cursor: pointer;
}
.form-example button:hover {
    background-color: #1c75bc;
}

Example Exercise

Try creating a form with custom styles:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>CSS Forms Exercise</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <div class="form-example">
        <form>
            <label for="name">Name:</label>
            <input type="text" id="name" name="name">
            <label for="email">Email:</label>
            <input type="email" id="email" name="email">
            <label for="password">Password:</label>
            <input type="password" id="password" name="password">
            <button type="submit">Submit</button>
        </form>
    </div>
</body>
</html>

Exercises

  1. Add a focus effect to the input fields to change the border color when they are clicked.
  2. Style the form button to change its background color and add a box shadow on hover.
  3. Create a disabled style for the button when the form is incomplete.
