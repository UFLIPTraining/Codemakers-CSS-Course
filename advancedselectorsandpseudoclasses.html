CSS Pseudo-classes

Learn how to use pseudo-classes in CSS to style elements based on their state or position in the DOM.

Example Exercise

Try using pseudo-classes:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>CSS Pseudo-classes Exercise</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <ul class="pseudo-class-example">
        <li>Item 1</li>
        <li>Item 2</li>
        <li>Item 3</li>
        <li>Item 4</li>
    </ul>
</body>
</html>

CSS Pseudo-elements

Discover how to use pseudo-elements in CSS to style parts of elements, such as the first line or first letter.

Example Exercise

Try using pseudo-elements:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>CSS Pseudo-elements Exercise</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <p class="pseudo-element-example">This is a paragraph to demonstrate pseudo-elements.</p>
</body>
</html>
