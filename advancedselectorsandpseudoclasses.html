CSS Advanced Selectors

Explore the different advanced selectors in CSS, including attribute selectors, combinators, and more.

Attribute Selectors

Attribute selectors allow you to select elements based on their attributes or attribute values. For example: 

input[type="text"] {
    border: 2px solid blue;
}

This will style all input elements with the type "text".

Combinators

Combinators are used to combine multiple selectors in various ways. For example: 

.parent > .child {
    background-color: lightgray;
}

This will style all elements with the class "child" that are direct children of elements with the class "parent".

CSS Pseudo-classes

Learn how to use pseudo-classes in CSS to style elements based on their state or position in the DOM.

:hover

The :hover pseudo-class is used to select elements when the user hovers over them. For example: 

a:hover {
    color: red;
}

This will change the color of a link to red when the user hovers over it.

:nth-child

The :nth-child() pseudo-class matches elements based on their position in a group of siblings. For example: 

li:nth-child(even) {
    background-color: #f4f4f4;
}

This will style every even-numbered list item with a light gray background.

CSS Pseudo-elements

Discover how to use pseudo-elements in CSS to style parts of elements, such as the first line or first letter.

::first-line

The ::first-line pseudo-element is used to apply styles to the first line of a block-level element. For example: 

p::first-line {
    font-weight: bold;
}

This will make the first line of a paragraph bold.

::before and ::after

The ::before and ::after pseudo-elements are used to insert content before or after an element's content. For example: 

p::before {
    content: "Note: ";
    color: red;
    font-weight: bold;
}

This will add "Note: " in red before the content of each paragraph.
