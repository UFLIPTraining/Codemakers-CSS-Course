CSS Flexbox

Flexbox is a one-dimensional layout system that allows you to create flexible and responsive layouts.

Flexbox Properties

/* CSS */
.flex-container {
  display: flex;
  justify-content: space-around;
  align-items: center;
  background-color: #232323;
  height: 200px;
}

.flex-item {
  background-color: #1c75bc;
  color: #fff;
  padding: 20px;
  margin: 10px;
  text-align: center;
}

More Flexbox Examples

Justify Content

The justify-content property aligns flex items along the main axis.

/* CSS */
.justify-content-container {
  display: flex;
  justify-content: space-between;
  background-color: lightgray;
  padding: 10px;
}

.justify-content-item {
  background-color: #1c75bc;
  color: white;
  padding: 10px;
}

Align Items

The align-items property aligns flex items along the cross axis.

/* CSS */
.align-items-container {
  display: flex;
  align-items: flex-start;
  background-color: lightgray;
  height: 200px;
  padding: 10px;
}

.align-items-item {
  background-color: #1c75bc;
  color: white;
  padding: 10px;
  margin: 5px;
}

Flex Direction

The flex-direction property defines the direction of the main axis.

/* CSS */
.flex-direction-container {
  display: flex;
  flex-direction: column;
  background-color: lightgray;
  padding: 10px;
}

.flex-direction-item {
  background-color: #1c75bc;
  color: white;
  padding: 10px;
  margin: 5px;
}

Order

The order property controls the order of flex items.

/* CSS */
.order-container {
  display: flex;
  background-color: lightgray;
  padding: 10px;
}

.order-item-1 {
  background-color: #1c75bc;
  color: white;
  padding: 10px;
  margin: 5px;
  order: 2;
}

.order-item-2 {
  background-color: #00cdff;
  color: white;
  padding: 10px;
  margin: 5px;
  order: 1;
}

.order-item-3 {
  background-color: #febd1d;
  color: white;
  padding: 10px;
  margin: 5px;
  order: 3;
}
