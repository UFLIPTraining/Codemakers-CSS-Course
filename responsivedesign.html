Responsive Web Design (RWD) Intro

Learn the basics of responsive web design, including the principles and techniques used to create responsive layouts.

Responsive Layout

To create a simple responsive layout, you can use the following example:

/* styles.css */
.responsive-container {
    display: flex;
    flex-wrap: wrap;
    gap: 20px;
}

.responsive-item {
    flex: 1 1 100px;
    background-color: #1c75bc;
    color: white;
    padding: 20px;
    box-sizing: border-box;
}

Start by creating a simple responsive layout:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>RWD Example</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <div class="responsive-container">
        <div class="responsive-item">Item 1</div>
        <div class="responsive-item">Item 2</div>
        <div class="responsive-item">Item 3</div>
    </div>
</body>
</html>

Exercises

  1. Create a new responsive layout with four items and different background colors for each item.
  2. Modify the example above to use a grid layout instead of Flexbox.
  3. Add media queries to change the layout to a single column on small screens.

RWD Viewport

Discover how to use the viewport meta tag and media queries to create responsive designs.

Viewport Meta Tag

The viewport meta tag is essential for responsive design. Here is an example:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Media Queries

Media queries allow you to apply CSS rules based on the size of the viewport. For example:

/* styles.css */
.viewport-example {
    background-color: #1c75bc;
    color: white;
    padding: 20px;
    text-align: center;
}

@media (max-width: 600px) {
    .viewport-example {
        background-color: #00cdff;
    }
}

Use the viewport meta tag and media queries to make a layout responsive:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Viewport and Media Queries</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <div class="viewport-example">
        Responsive Design Example
    </div>
</body>
</html>

Exercises

  1. Create a responsive layout that changes the background color of a section based on three different viewport sizes.
  2. Add a viewport meta tag to an existing HTML file and create media queries to adjust the font size for different screen widths.
  3. Create a responsive navigation menu that changes layout between horizontal and vertical based on screen width.

RWD Grid View

Explore how to create responsive grid layouts using CSS Grid and Flexbox.

CSS Grid

CSS Grid is a powerful tool for creating responsive layouts. Here is an example:

/* styles.css */
.grid-container {
    display: grid;
    grid-template-columns: repeat(auto-fit, minmax(100px, 1fr));
    gap: 20px;
}

.grid-item {
    background-color: #1c75bc;
    color: white;
    padding: 20px;
    text-align: center;
}

Create a responsive grid layout:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Responsive Grid</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <div class="grid-container">
        <div class="grid-item">Grid Item 1</div>
        <div class="grid-item">Grid Item 2</div>
        <div class="grid-item">Grid Item 3</div>
    </div>
</body>
</html>

Exercises

  1. Create a responsive grid layout with six items, adjusting the number of columns based on screen width.
  2. Modify the grid example to include different row heights using the grid-template-rows property.
  3. Add media queries to change the grid layout to a single column on small screens and three columns on larger screens.

RWD Media Queries

Learn how to use media queries to create responsive designs that adapt to different screen sizes.

Responsive Media Queries

Implement media queries to adapt the layout:

/* styles.css */
.media-query-example {
    background-color: #1c75bc;
    color: white;
    padding: 20px;
    text-align: center;
}

@media (max-width: 768px) {
    .media-query-example {
        background-color: #00cdff;
    }
}

@media (max-width: 480px) {
    .media-query-example {
        background-color: #febd1d;
    }
}

Implement media queries to adapt the layout:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Responsive Media Queries</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <div class="media-query-example">
        Media Query Example
    </div>
</body>
</html>

Exercises

  1. Create a responsive layout that changes the font size and text color based on different screen widths using media queries.
  2. Implement media queries to hide certain elements on small screens and display them on larger screens.
  3. Create a responsive card layout that adjusts the number of columns and the card sizes based on screen width.
