Responsive Design

Responsive web design (RWD) ensures that web pages look good on all devices. It is achieved using flexible layouts, flexible images, and CSS media queries.

Responsive Layout

/* CSS */
.responsive-container {
  display: flex;
  flex-wrap: wrap;
  gap: 20px;
}

.responsive-item {
  flex: 1 1 100px;
  background-color: #1c75bc;
  color: white;
  padding: 20px;
  box-sizing: border-box;
}

Responsive Images

Use the max-width property to make images responsive. This ensures that images scale down as the screen size decreases.

/* CSS */
.responsive-image {
  max-width: 100%;
  height: auto;
}

Inspect the image above and try changing its max-width and height properties.

Responsive Videos

To make videos responsive, use a wrapper element with a percentage-based padding to maintain the aspect ratio.

/* CSS */
.video-container {
  position: relative;
  padding-bottom: 56.25%; /* 16:9 aspect ratio */
  height: 0;
  overflow: hidden;
  max-width: 100%;
  background: #000;
}

.video-container iframe,
.video-container object,
.video-container embed {
  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
}

Inspect the video above and try changing its width and height properties within the wrapper.

Responsive Media Queries

Media queries allow you to apply different styles for different screen sizes. This is essential for responsive design.

/* CSS */
@media (max-width: 768px) {
  .responsive-media-example {
    background-color: lightblue;
  }
}

@media (max-width: 480px) {
  .responsive-media-example {
    background-color: lightgreen;
  }
}
Resize the browser to see the background color change.
