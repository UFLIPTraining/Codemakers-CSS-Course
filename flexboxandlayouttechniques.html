CSS Flexbox

Learn how to use CSS Flexbox to create flexible and responsive layouts. Flexbox is a layout model that allows you to design complex layout structures with ease.

Basic Flexbox Layout

Here's a basic example of using Flexbox:

/* styles.css */
.flex-container {
    display: flex;
    background-color: #333;
}

.flex-item {
    background-color: #1c75bc;
    color: white;
    margin: 10px;
    padding: 20px;
    font-size: 30px;
    text-align: center;
}

Here's a basic example of using Flexbox:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Basic Flexbox Layout</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <div class="flex-container">
        <div class="flex-item">Item 1</div>
        <div class="flex-item">Item 2</div>
        <div class="flex-item">Item 3</div>
    </div>
</body>
</html>

Exercises

  1. Create a Flexbox layout with 4 items. Experiment with different flex properties such as justify-content and align-items.
  2. Change the direction of the flex items using the flex-direction property.
  3. Adjust the size of one of the flex items using the flex-grow, flex-shrink, and flex-basis properties.

CSS Grid

Explore the CSS Grid layout system and how to use it to create complex and responsive grid layouts. CSS Grid is a powerful layout system that allows you to create two-dimensional layouts.

Basic Grid Layout

Here's a basic example of using CSS Grid:

/* styles.css */
.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 10px;
    background-color: #333;
    padding: 10px;
}

.grid-item {
    background-color: #1c75bc;
    color: white;
    border: 1px solid #444;
    padding: 20px;
    font-size: 30px;
    text-align: center;
}

Here's a basic example of using CSS Grid:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Basic Grid Layout</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <div class="grid-container">
        <div class="grid-item">1</div>
        <div class="grid-item">2</div>
        <div class="grid-item">3</div>
        <div class="grid-item">4</div>
        <div class="grid-item">5</div>
        <div class="grid-item">6</div>
    </div>
</body>
</html>

Exercises

  1. Create a Grid layout with 4 columns and 2 rows. Style the grid items differently from the example above.
  2. Adjust the grid-template-columns to use different units (e.g., percentages, fixed widths).
  3. Experiment with spanning multiple rows and columns with grid items using the grid-column and grid-row properties.
