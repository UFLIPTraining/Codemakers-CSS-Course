Using the Inspect Tool

The browser's inspect tool is a powerful feature that allows you to see and edit HTML and CSS on the fly. This is incredibly useful for testing changes and debugging your code.

How to Use the Inspect Tool

Right-click on any element on the page and select "Inspect" to open the developer tools. Here, you can see the HTML and CSS that make up the page.

Inspect this element and try changing the border color and padding in the styles pane.

div.example {
  border: 1px solid red;
  padding: 10px;
}

Try changing the border to border: 1px solid blue; and the padding to padding: 20px;.

Example 1: Changing Background Color

Inspect this element and change the background color.

div.example-bg {
  background-color: lightgray;
  padding: 10px;
}

Try changing the background color to background-color: yellow;.

Example 2: Modifying Font Size

Inspect this element and change the font size.

div.example-font {
  font-size: 16px;
}

Try changing the font size to font-size: 24px;.

Example 3: Adjusting Margin

Inspect this element and change the margin.

div.example-margin {
  margin: 20px;
  background-color: #1c75bc;
  color: white;
}

Try changing the margin to margin: 40px;.

Example 4: Adding Box Shadow

Inspect this element and change the box shadow.

div.example-shadow {
  padding: 10px;
  box-shadow: 2px 2px 5px rgba(0, 0, 0, 0.3);
}

Try changing the box shadow to box-shadow: 4px 4px 10px rgba(0, 0, 0, 0.5);.

Example 5: Changing Font Style

Inspect this element and change the font style.

div.example-font-style {
  font-style: normal;
}

Try changing the font style to font-style: italic;.

Example 6: Adding Border Radius

Inspect this element and change the border radius.

div.example-radius {
  border: 1px solid #1c75bc;
  padding: 10px;
  border-radius: 0;
}

Try changing the border radius to border-radius: 10px;.

Example 7: Adjusting Line Height

Inspect this element and change the line height.

div.example-line-height {
  line-height: 1.2;
}

Try changing the line height to line-height: 2;.
