Using the Inspect Tool

The browser's inspect tool is a powerful feature that allows you to see and edit HTML and CSS on the fly. This is incredibly useful for testing changes and debugging your code.

How to Use the Inspect Tool

Right-click on any element on the page and select "Inspect" to open the developer tools. Here, you can see the HTML and CSS that make up the page.

Inspect this element and try changing the border color and padding in the styles pane.

div.example {
  border: 1px solid red;
  padding: 10px;
}

Try changing the border to border: 1px solid blue; and the padding to padding: 20px;.

Exercises

  1. Change the background color of the div.example element to lightgray using the inspect tool.
  2. Add a new CSS property margin: 10px; to the div.example element using the inspect tool and observe the changes.
  3. Modify the text inside the div.example element directly from the inspect tool to say "This text has been changed!"

Example 1: Changing Background Color

Inspect this element and change the background color.

div.example-bg {
  background-color: lightgray;
  padding: 10px;
}

Try changing the background color to background-color: yellow;.

Exercises

  1. Create a new element with the class example-bg-2 and set its background color to green.
  2. Change the background color of the example-bg element to red and observe the changes.
  3. Add a new CSS class called hover-bg-color that changes the background color to blue on hover. Apply this class to one of the existing elements.
