Borders

The border property allows you to specify the style, width, and color of an element's border.

/* CSS */
.border-example {
  border: 1px solid red;
  padding: 10px;
  margin-bottom: 10px;
}

.border-example-2 {
  border: 2px dashed green;
  padding: 10px;
  margin-bottom: 10px;
}

.border-example-3 {
  border: 3px dotted blue;
  padding: 10px;
  margin-bottom: 10px;
}
1px solid red border
2px dashed green border
3px dotted blue border

Inspect the elements above and try changing their border properties.

Colors

The color property sets the color of the text, while the background-color property sets the background color of an element.

/* CSS */
.color-example {
  color: red;
  background-color: yellow;
  padding: 10px;
  margin-bottom: 10px;
}

.color-example-2 {
  color: blue;
  background-color: lightgrey;
  padding: 10px;
  margin-bottom: 10px;
}

.color-example-3 {
  color: green;
  background-color: pink;
  padding: 10px;
  margin-bottom: 10px;
}
Red text on yellow background
Blue text on lightgrey background
Green text on pink background

Inspect the elements above and try changing their color and background-color properties.
