CSS Colors

Explore the different color options available in CSS, including color names, hex codes, RGB, and HSL values.

Color Names

CSS supports 140 standard color names. For example:

p {
    color: red;
}

This will make the text in the paragraph red.

Hex Codes

Hexadecimal color codes are six-digit codes representing the intensity of red, green, and blue. For example: 

p {
    color: #ff0000;
}

This will make the text in the paragraph red.

RGB

RGB values are specified with the rgb() function, which defines the intensity of red, green, and blue. For example: 

p {
    color: rgb(255, 0, 0);
}

This will make the text in the paragraph red.

HSL

HSL stands for hue, saturation, and lightness. For example:

p {
    color: hsl(0, 100%, 50%);
}

This will make the text in the paragraph red.

CSS Backgrounds

Learn how to set and style backgrounds in CSS, including background color, images, and gradients.

Background Color

You can set a background color for an element using the background-color property. For example: 

div {
    background-color: lightblue;
}

This will set the background color of the div to light blue.

Background Image

You can set a background image for an element using the background-image property. For example: 

div {
    background-image: url('background.jpg');
}

This will set the background image of the div to the specified image.

Background Gradient

CSS gradients let you display smooth transitions between two or more specified colors. For example: 

div {
    background: linear-gradient(to right, red, yellow);
}

This will create a gradient background that transitions from red to yellow.
