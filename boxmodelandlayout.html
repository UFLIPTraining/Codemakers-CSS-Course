CSS Box Model

Understand the CSS box model and how to use it to create layouts, including margin, border, padding, and content areas.

Box Model Explanation

The CSS box model represents the structure of an element and consists of margins, borders, padding, and the actual content. Here is an example:


/* styles.css */
.box-model-example {
    width: 300px;
    padding: 20px;
    border: 5px solid #00cdff;
    margin: 15px;
    background-color: #fff;
}

Example Exercise

Try creating a div with the following CSS properties to see how the box model works:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>CSS Box Model Exercise</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <div class="box-model-example">This is a box model example.</div>
</body>
</html>

CSS Display

Learn about the different display properties in CSS and how to use them to control the layout of your elements.

Display Properties

CSS has several display properties, such as block, inline, inline-block, and none. Here is an example:


/* styles.css */
.display-example-block {
    display: block;
    width: 100%;
    background-color: #00cdff;
    padding: 10px;
    margin-bottom: 10px;
}

.display-example-inline {
    display: inline;
    background-color: #febd1d;
    padding: 10px;
    margin-right: 10px;
}

.display-example-inline-block {
    display: inline-block;
    background-color: #e55440;
    padding: 10px;
    margin-right: 10px;
}

Example Exercise

Try creating multiple divs with different display properties to see how they behave:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>CSS Display Exercise</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <div class="display-example-block">Block Element</div>
    <div class="display-example-inline">Inline Element</div>
    <div class="display-example-inline">Inline Element</div>
    <div class="display-example-inline-block">Inline-Block Element</div>
    <div class="display-example-inline-block">Inline-Block Element</div>
</body>
</html>

CSS Flexbox

Discover how to use CSS Flexbox to create flexible and responsive layouts.

Flexbox Properties

Flexbox is a powerful layout module that allows you to create complex layouts with ease. Here is an example:


/* styles.css */
.flex-container {
    display: flex;
    justify-content: space-around;
    align-items: center;
    background-color: #232323;
    height: 200px;
}

.flex-item {
    background-color: #1c75bc;
    color: #fff;
    padding: 20px;
    margin: 10px;
    text-align: center;
}

Example Exercise

Try creating a flex container with multiple flex items:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>CSS Flexbox Exercise</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <div class="flex-container">
        <div class="flex-item">Flex Item 1</div>
        <div class="flex-item">Flex Item 2</div>
        <div class="flex-item">Flex Item 3</div>
    </div>
</body>
</html>

CSS Grid

Explore the CSS Grid layout system and how to use it to create complex and responsive grid layouts.

Grid Properties

CSS Grid is a two-dimensional layout system that allows you to create complex layouts with ease. Here is an example:


/* styles.css */
.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 10px;
    background-color: #232323;
    padding: 10px;
}

.grid-item {
    background-color: #1c75bc;
    color: #fff;
    padding: 20px;
    text-align: center;
}
