CSS Box Model

Understand the CSS box model and how to use it to create layouts, including margin, border, padding, and content areas.

Box Model Explanation

The CSS box model represents the structure of an element and consists of margins, borders, padding, and the actual content. Here is an example:


/* styles.css */
.box-model-example {
  width: 300px;
  padding: 20px;
  border: 5px solid #00cdff;
  margin: 15px;
  background-color: #fff;
}

Example Exercise

Try creating a div with the following CSS properties to see how the box model works:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>CSS Box Model Exercise</title>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <div class="box-model-example">This is a box model example.</div>
</body>
</html>

Exercises

  1. Create a new div with different padding and margin values and observe the changes.
  2. Adjust the border width and color of the div and see how it affects the layout.
  3. Use the box-sizing: border-box; property and see how it changes the box model behavior.

CSS Display

Learn about the different display properties in CSS and how to use them to control the layout of your elements.

Display Properties

CSS has several display properties, such as block, inline, inline-block, and none. Here is an example:


/* styles.css */
.display-example-block {
  display: block;
  width: 100%;
  background-color: #00cdff;
  padding: 10px;
  margin-bottom: 10px;
}

.display-example-inline {
  display: inline;
  background-color: #febd1d;
  padding: 10px;
  margin-right: 10px;
}

.display-example-inline-block {
  display: inline-block;
  background-color: #e55440;
  padding: 10px;
  margin-right: 10px;
}

Example Exercise

Try creating multiple divs with different display properties to see how they behave:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>CSS Display Exercise</title>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <div class="display-example-block">Block Element</div>
  <div class="display-example-inline">Inline Element</div>
  <div class="display-example-inline">Inline Element</div>
  <div class="display-example-inline-block">Inline-Block Element</div>
  <div class="display-example-inline-block">Inline-Block Element</div>
</body>
</html>

Exercises

  1. Create an inline element and observe how it affects the flow of content around it.
  2. Change the display property of an element from block to inline-block and see the difference.
  3. Hide an element using the display: none; property and observe its impact on the layout.

CSS Flexbox

Discover how to use CSS Flexbox to create flexible and responsive layouts.

Flexbox Properties

Flexbox is a powerful layout module that allows you to create complex layouts with ease. Here is an example:


/* styles.css */
.flex-container {
  display: flex;
  justify-content: space-around;
  align-items: center;
  background-color: #232323;
  height: 200px;
}

.flex-item {
  background-color: #1c75bc;
  color: #fff;
  padding: 20px;
  margin: 10px;
  text-align: center;
}

Example Exercise

Try creating a flex container with multiple flex items:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>CSS Flexbox Exercise</title>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <div class="flex-container">
    <div class="flex-item">Flex Item 1</div>
    <div class="flex-item">Flex Item 2</div>
    <div class="flex-item">Flex Item 3</div>
  </div>
</body>
</html>

Exercises

  1. Adjust the justify-content property to align items to the start, center, and end of the container.
  2. Change the flex-direction property to column and observe how the layout changes.
  3. Use the align-items property to align items to the start, center, and end of the cross-axis.

CSS Grid

Explore the CSS Grid layout system and how to use it to create complex and responsive grid layouts.

Grid Properties

CSS Grid is a two-dimensional layout system that allows you to create complex layouts with ease. Here is an example:


/* styles.css */
.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 10px;
  background-color: #232323;
  padding: 10px;
}

.grid-item {
  background-color: #1c75bc;
  color: #fff;
  padding: 20px;
  text-align: center;
}

Example Exercise

Try creating a grid container with multiple grid items:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>CSS Grid Exercise</title>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <div class="grid-container">
    <div class="grid-item">Grid Item 1</div>
    <div class="grid-item">Grid Item 2</div>
    <div class="grid-item">Grid Item 3</div>
    <div class="grid-item">Grid Item 4</div>
    <div class="grid-item">Grid Item 5</div>
    <div class="grid-item">Grid Item 6</div>
  </div>
</body>
</html>

Exercises

  1. Create a grid layout with 4 columns and 2 rows, and place items in specific grid areas using the grid-area property.
  2. Use the grid-template-rows property to define different row heights and observe the changes.
  3. Adjust the gap property to change the spacing between grid items.
