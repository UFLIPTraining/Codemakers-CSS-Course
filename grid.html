CSS Grid

CSS Grid is a two-dimensional layout system that allows you to create complex and responsive layouts.

Grid Properties

/* CSS */
.grid-container {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(100px, 1fr));
  gap: 10px;
  background-color: #232323;
  padding: 10px;
}

.grid-item {
  background-color: #1c75bc;
  color: #fff;
  padding: 20px;
  text-align: center;
}

More Grid Examples

Grid Template Rows

The grid-template-rows property defines the rows of the grid.

/* CSS */
.grid-rows-container {
  display: grid;
  grid-template-columns: 1fr 1fr;
  grid-template-rows: 100px 200px;
  gap: 10px;
  background-color: lightgray;
  padding: 10px;
}

.grid-rows-item {
  background-color: #1c75bc;
  color: white;
  padding: 20px;
  text-align: center;
}

Grid Area

The grid-area property defines a grid itemâ€™s position within the grid by specifying the start and end lines for both rows and columns.

/* CSS */
.grid-area-container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  grid-template-rows: 100px 100px;
  gap: 10px;
  background-color: lightgray;
  padding: 10px;
}

.grid-area-item {
  background-color: #1c75bc;
  color: white;
  padding: 20px;
  text-align: center;
}

.item1 {
  grid-area: 1 / 1 / 2 / 3; /* row-start / column-start / row-end / column-end */
}

.item2 {
  grid-area: 1 / 3 / 3 / 4;
}

.item3 {
  grid-area: 2 / 1 / 3 / 3;
}
