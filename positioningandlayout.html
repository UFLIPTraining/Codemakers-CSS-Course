CSS Positioning

Discover the different positioning techniques in CSS, including static, relative, absolute, fixed, and sticky positioning.

Example Exercise

Try using different positioning techniques:

/* styles.css */
.static-position {
  position: static; /* Default position */
}

.relative-position {
  position: relative;
  top: 20px;
  left: 20px;
}

.absolute-position {
  position: absolute;
  top: 50px;
  left: 50px;
}

.fixed-position {
  position: fixed;
  top: 10px;
  right: 10px;
}

.sticky-position {
  position: -webkit-sticky; /* For Safari */
  position: sticky;
  top: 0;
  background-color: yellow;
  padding: 50px;
  font-size: 20px;
}

CSS Floats

Learn how to use floats in CSS to create layouts and how to clear floats properly.

Example Exercise

Try creating a layout using floats:

/* styles.css */
.float-container {
  width: 100%;
}

.float-left {
  float: left;
  width: 45%;
  background-color: lightblue;
  padding: 20px;
}

.float-right {
  float: right;
  width: 45%;
  background-color: lightgreen;
  padding: 20px;
}

CSS Clearfix

Understand the clearfix technique and how to use it to clear floats in CSS.

Example Exercise

Try using the clearfix technique:

/* styles.css */
.clearfix::after {
  content: "";
  clear: both;
  display: table;
}

.clearfix-container {
  background-color: lightgrey;
}

.float-left, .float-right {
  float: left; /* Ensure both floats are left-aligned for this example */
  width: 45%;
  padding: 20px;
}
