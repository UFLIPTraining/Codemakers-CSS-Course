CSS Grid Introduction

Learn the basics of CSS Grid, including grid containers, grid items, and grid lines.

Simple Grid Layout

To create a simple grid layout, you can use the following example:

/* styles.css */
.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 10px;
  background-color: #333;
  padding: 10px;
}

.grid-item {
  background-color: #1c75bc;
  color: white;
  border: 1px solid #444;
  padding: 20px;
  font-size: 30px;
  text-align: center;
}

Start by creating a simple grid layout:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>CSS Grid Example</title>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <div class="grid-container">
    <div class="grid-item">1</div>
    <div class="grid-item">2</div>
    <div class="grid-item">3</div>
    <div class="grid-item">4</div>
    <div class="grid-item">5</div>
    <div class="grid-item">6</div>
  </div>
</body>
</html>

Exercises

  1. Create a grid container with 4 columns and 2 rows. Place 8 grid items inside it and style them differently from the example above.
  2. Adjust the gap between the grid items to 20px and change the background color of the grid container.
  3. Experiment with different values for grid-template-columns (e.g., repeat(4, 1fr), 1fr 2fr 1fr).

Grid Container with Rows and Columns

Create a grid container with specified rows and columns:

/* styles.css */
.grid-container {
  display: grid;
  grid-template-columns: 1fr 2fr;
  grid-template-rows: auto;
  gap: 10px;
  background-color: #333;
  padding: 10px;
}

.grid-item {
  background-color: #1c75bc;
  color: white;
  border: 1px solid #444;
  padding: 20px;
  font-size: 30px;
  text-align: center;
}

CSS Grid Item

Explore how to position and style grid items in CSS Grid.

Position Grid Items

Position grid items within a grid container:

/* styles.css */
.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows: 100px 100px;
  gap: 10px;
  background-color: #333;
  padding: 10px;
}

.grid-item {
  background-color: #1c75bc;
  color: white;
  border: 1px solid #444;
  padding: 20px;
  font-size: 30px;
  text-align: center;
}

.item1 {
  grid-column: 1 / span 2;
}

.item2 {
  grid-column: 3;
  grid-row: 1 / span 2;
}

Position grid items within a grid container:

Exercises

  1. Create a grid container and position items using the grid-column and grid-row properties.
  2. Use the grid-area property to place grid items in specific areas of the grid container.
  3. Experiment with spanning multiple rows and columns with grid items.
