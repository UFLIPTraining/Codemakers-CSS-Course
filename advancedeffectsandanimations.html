CSS Transitions

Learn how to create smooth transitions in CSS to enhance the user experience.

Transition Property

The transition property is used to create a smooth transition effect between different states of an element. For example:

button {
  background-color: #1c75bc;
  transition: background-color 0.3s ease;
}

button:hover {
  background-color: #00cdff;
}

This will create a smooth background color transition on hover.

Exercises

  1. Create a button that changes its width smoothly on hover.
  2. Implement a transition effect that changes the opacity of a div when hovered over.
  3. Apply a transition effect that changes the color and background color of a link on hover.

CSS Animations

Discover how to create animations in CSS to bring your web pages to life.

@keyframes Rule

The @keyframes rule is used to define the animation. For example:

@keyframes example {
  0% {
    transform: translateX(0);
  }
  100% {
    transform: translateX(100px);
  }
}

.animation-example {
  animation: example 2s infinite;
}

This will create an animation that moves an element 100px to the right.

Exercises

  1. Create an animation that changes the background color of a div from red to blue.
  2. Implement an animation that rotates an element 360 degrees.
  3. Create a keyframe animation that scales an element up and down continuously.

CSS Transformations

Explore the different transformation properties in CSS, including scaling, rotating, and skewing elements.

Transform Property

The transform property is used to apply a 2D or 3D transformation to an element. For example:

div {
  transform: rotate(45deg);
}

This will rotate the div 45 degrees.

Exercises

  1. Create a transformation that scales a div by 1.5 times its original size.
  2. Apply a skew transformation to an element to skew it 20 degrees along the X-axis.
  3. Combine multiple transformations (rotate, scale, and translate) on a single element.
