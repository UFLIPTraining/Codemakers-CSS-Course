Pseudo Classes

Pseudo-classes are keywords added to selectors that specify a special state of the selected elements.

Common Pseudo Classes

/* CSS */
.pseudo-class-example:hover {
  color: red;
}

.pseudo-class-example:focus {
  background-color: yellow;
}

.pseudo-class-example:active {
  font-weight: bold;
}

Nth-child Pseudo Class

The :nth-child() pseudo-class matches elements based on their position in a group of siblings.

/* CSS */
.nth-child-example:nth-child(odd) {
  background-color: #1c75bc;
  color: white;
}

.nth-child-example:nth-child(even) {
  background-color: #00cdff;
  color: black;
}

Not Pseudo Class

The :not() pseudo-class selects elements that do not match a certain selector.

/* CSS */
.not-example:not(.exclude) {
  background-color: lightcoral;
  color: white;
  padding: 10px;
  margin: 5px;
}

First-child and Last-child Pseudo Classes

The :first-child and :last-child pseudo-classes target the first and last child elements within a parent.

/* CSS */
.first-child-example:first-child {
  background-color: #1c75bc;
  color: white;
}

.last-child-example:last-child {
  background-color: #00cdff;
  color: black;
}
