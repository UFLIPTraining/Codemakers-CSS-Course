CSS Variables

Learn how to use CSS variables to store values and reuse them throughout your CSS.

Example: Using CSS Variables

/* styles.css */
:root {
  --main-bg-color: #1c75bc;
  --main-text-color: #ffffff;
}

.variable-example {
  background-color: var(--main-bg-color);
  color: var(--main-text-color);
  padding: 20px;
  font-size: 18px;
}

Here's an example HTML using CSS Variables:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>CSS Variables Example</title>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <div class="variable-example">This is a sample div.</div>
</body>
</html>

Exercises

  1. Create a CSS variable for the font size and apply it to a paragraph.
  2. Define CSS variables for primary and secondary colors and use them to style different elements.
  3. Create a theme switcher that changes CSS variables on button click to switch between light and dark modes.

CSS Math Functions

Discover how to use CSS math functions, such as calc(), to perform calculations in your CSS.

Example: Using calc()

/* styles.css */
.calc-example {
  width: calc(100% - 40px);
  padding: 20px;
  background-color: #1c75bc;
  color: white;
  text-align: center;
}

Here's an example HTML using calc():

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>CSS Math Functions Example</title>
  <link rel="stylesheet" href="styles.css">
</head>
<body>
  <div class="calc-example">This is a sample div.</div>
</body>
</html>

Exercises

  1. Use calc() to set the height of an element to be half the viewport height minus 20px.
  2. Create a responsive layout that uses calc() to adjust margins and paddings based on the screen size.
  3. Implement a flexible grid system using calc() to set the width of the columns.
